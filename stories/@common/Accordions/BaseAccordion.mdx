## 更多例子

```tsx
// 禁止展开
<BaseAccordion title="Title" disabled>xxx</BaseAccordion>

// 默认展开
<BaseAccordion title="Title" defaultExpanded>xxx</BaseAccordion>

// 反转展开图标的位置
<BaseAccordion title="Title" reverseExpandIcon>xxx</BaseAccordion>

// 自定义展开图标
<BaseAccordion title="Title" expandIcon={"A"}>xxx</BaseAccordion>

// 自定义展开内容右下角的扩展内容
<BaseAccordion
    title="Title"
    actions={
        <>
            <button>AA<button>
            <button>BB<button>
        </>
    }
>
    xxx
</BaseAccordion>
```
